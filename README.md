# Task Manager Web App

A simple and responsive task manager application built using HTML, CSS, and vanilla JavaScript. Tasks are saved using localStorage.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Data persistence using localStorage
- Responsive design for mobile and desktop

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser localStorage
